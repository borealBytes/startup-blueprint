# Perplexity Space Setup

Use this repo as the source of truth for agentic standards.

## Setup Steps

1. Paste `docs/agentic/instructions.md` into the Space instructions.
2. Upload the remaining files from `docs/agentic/` (including `docs/agentic/adr/`).
3. When these docs change in the repo, re-upload the updated files to keep Space in sync.
